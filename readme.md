LSDTopoTools Chi Tools from Mudd et al 2014
================================================


This folder contains source code for the Chi analysis tools described in [Mudd et al (2014), open access](http://onlinelibrary.wiley.com/doi/10.1002/2013JF002981/full).

This code is part of the [LSDTopoTools software package](http://lsdtopotools.github.io/).

The versions here are updated on a semi regular basis. 

For instructions, go to the [LSDTopoTools documentation website](http://www.geos.ed.ac.uk/~smudd/LSDTT_docs/html/chi_profiles.html).
